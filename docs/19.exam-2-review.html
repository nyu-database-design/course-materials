``` -- Calculate average of all values in the `salary_range_to` field, grouping by `agency`: ```javascript db.collection.aggregate([ { $group: { _id: "$agency", avg_val: { $avg: "$salary_range_to" }, }, }, ]); ``` -- Other statistics are calculated very similarly. --- name: web-apps # Web Apps -- ## Key points Main topics: -- - the web is the use of the `HTTP` protocol that allows a web browser to browse _hypertext_ documents -- - a client (e.g. web browser) makes requests (e.g. `HTTP` `GET` or `POST` requests), and a server (e.g. web server) issues responses to those requests (e.g. `HTTP` `200` `OK`). -- - `HTTP` `POST` requests can contain a payload of body data, such as when the user submits a form. `GET` requests (the default type) cannot. -- - `flask` is a Python web server module to help detect incoming requests and respond to them -- - `flask` allows us to set up server-side _routes_ - functions that run automatically when a browser requests a certain web address. -- - `flask` also makes it easy to generate templates: static `HTML` documents with data injected into them dynamically. -- - `pymongo` is a Python module that helps the server code communicate with a MongoDB databases. --- template: web-apps ## HTML, CSS, Javascript Web browsers can only "understand"/interpret 3 languages: `HTML`, `CSS`, Javascript`. -- - HTML is used to indicate the content and semantic meaning of the content of a given web page. -- - CSS is used to indicate the styling of the content on a given web page. -- - Javascript is used to indicate any interactive behaviors of a given web page (changes to make on user `click`, `mouseover`, `keyPressed`, etc). --- name: pandas # Pandas -- ## Key points Main topics: -- - powerful Python-based data analysis library -- - stores data in `Series` and `DataFrame` data structures -- - vector math! -- - relies on `numpy`, wraps around `matplotlib` -- - includes tools to help munge/cleanup the data. -- - supports basic data analysis: statistical analysis including grouping. --- name: conclusions # Conclusions -- Thank you.
